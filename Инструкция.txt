Инструкция.

Положить все файлы в один каталог.
Запустить командную строку от администратора.
Командой cd перейти в нужный каталог.
Запустить программу, например следующим образом:
java FileFilter -s -a -p sample- in1.txt in2.txt

Опции:
-o путь до результатов
-p Префикс имен выходных файлов
-a режим добавления в существующие файлы
-s краткая статистика
-f полная статистика
